    • This is the project I worked on for my dissertation. I loved the idea of randomly generating Dark-Souls or Zelda levels for more replayability but there are few games that do this in the action-adventure genre. So my dissertation set out to create a basic (no environmental features) level generator for action-adventure style games which included generating lock&key style structures, as seen in Zelda or metroidvanias. By using techniques such as scaling distribution, cellular automata, marching squares and Dorman's Cyclic patterns, it manages to do just that. Notable features:

  • C# Unity scripting-centred approach for all areas of the project
  • Souls-like character controls and movement, including dodging, sprinting and invincible frames
  • 2D and 3D map and mesh generation using cellular automata and marching squares
  • Full use of Unity's animation systems, controllers, events etc.
  • Multidirectional animation (running/rolling) using blend trees, layers
  • Lock on targeting with UI and event system interaction
  • Other UI elements such as: Health bars, UI camera scripts, Lock-on crosshair's etc.
  • Weapon-based animations with Collider & animation events, hitbox physics
  • Seven weapon types all equipable by both enemies and character in a easy slot in/out system utilizing proper inheritance
  • Stat-based combat/gameplay(str,def,speed effects combat)
  • Node Graph map generation and manipulation (including graph traversal)
  • Ragdoll death physics
